## 전각 반각 

```
KSC5601 (KS C 5601) 또는 KS X 1001은 한글과 한자를 표현하는 코드 체계로, 1976년에 처음 제정되었습니다. 이 표준은 두 바이트로 구성되어 있으며, 2,350자의 완성형 한글과 4,888자의 한자를 포함합니다.
1. 코드 구성

    문자 세트: 2바이트 구성, 94자의 조합형 자음 및 모음 지원
    최상위 비트(MSB): 1이며, 코드 범위는 0xA1~0xFE
    표현 가능 문자 수: 최대 8,836자 (완성형 한글 2,350자, 한자 4,888자, 특수문자 432자 등)

2. 코드 구분

    A1A1~A2FE: 구두점 및 기호 (예: ※, ☎)
    A3A1~A3FE: 전각 ASCII (예: A, B, *)
    A4A1~A4FE: 2벌식 한글 자음 및 모음 (예: ㄱ, ㄲ)
    A5A1~A9FE: 로마 숫자, 그리스 문자, 기타 기호
    AAA1~ABFE: 일본어 (히라가나, 카타카나)
    ACA1~ACFE: 소련어
    BOA1~C8FE: 한글 음절 (예: 가, 각)
    CAA1~FDFE: 한자

이 표준은 정보 교환을 위한 부호 체계로, 한국표준협의회에 의해 제정되었습니다.

```

## Java 업그레이드 안내 (Java 21)

이 저장소는 Java 21(LTS)로 컴파일 및 실행될 수 있도록 확인되었습니다. 시스템에 Java 21이 설치되어 있는지 확인하려면:

```bash
java -version
javac -version
```

컴파일(모든 소스) 예시:

```bash
mkdir -p out21
javac --release 21 -d out21 -cp "lib/*" $(find src -name "*.java")
```

간단 실행 예시(예: `json.Main`):

```bash
java -cp out21:lib/* json.Main
```

문제 발생 시:
- 외부 라이브러리가 있다면 `lib/`에 JAR을 두고 클래스패스에 포함하세요.
- 특정 컴파일 오류가 있으면 `--release 21` 없이 먼저 `javac -d out21 ...`로 컴파일 해보세요.
